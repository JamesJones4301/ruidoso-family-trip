# Ruidoso Family Trip Site

Simple static website for the Ruidoso family trip itinerary.

## Files
- `index.html` contains the mobile-friendly trip website.
- `vercel.json` provides simple Vercel settings.
- `package.json` pins Node 24 for cleaner Vercel compatibility.

## Deploy on Vercel
1. Import this repository into Vercel.
2. Keep the default root since this repo contains only the trip site.
3. Deploy.
